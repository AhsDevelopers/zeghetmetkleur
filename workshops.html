3D

Zeg het met 3D

Tijdens deze workshop maak je kennis met 3D-printing. Je leert de werking van een 3D-printer kennen. Je gaat zelf aan de slag met een 3D-printer. Je leert zelf een object modelleren en klaarmaken om te 3D-printen. We bespreken en analyseren interessante voorbeelden en toepassingen.

Verantwoordelijke docent
Annelien Suenaert

Code

Start to Code: Web of Things

Deze workshop zoomt in op het leren programmeren door software met hardware te combineren. Om hardware toegankelijk te maken zullen we gebruik maken van de “Raspberry Pi” met daarop een module “Sense Hat”. De software wordt geschreven m.b.v. de programmeertaal “Python” en de cloud service “Google Firebase”.
