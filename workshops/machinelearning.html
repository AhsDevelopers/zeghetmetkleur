Machine Learning

Machine learning technologie leert computers hoe ze taken moeten uitvoeren door te leren van data, in plaats van ze te programmeren. Machine learning leert dus dankzij geavanceerde algoritmen die voortkomen uit immense hoeveelheden big data. Hoe meer data toegankelijk zijn voor algoritmen, hoe meer computers kunnen leren. Denk bijvoorbeeld aan productaanbevelingen op websites, gezichtsherkenning en de suggesties voor de snelste route in routeplanners.
